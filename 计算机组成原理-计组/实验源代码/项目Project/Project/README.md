# 如何运行

运行 Mars4_5.jar

![image-20240531133146068.png](https://s2.loli.net/2024/05/31/6eBMJcw3VxEfyFk.png)



点击左上角File，再点击Open

![image-20240531133253302.png](https://s2.loli.net/2024/05/31/o9uOYBEP3qUzijf.png)



选择Project_MIPS.asm，再点击打开

![Snipaste_2024-05-31_13-45-34.png](https://s2.loli.net/2024/05/31/9eITY8qsrty3lAP.png)



打开后即可见软件代码

![Snipaste_2024-05-31_13-47-23.png](https://s2.loli.net/2024/05/31/z47QiEFUT3jurl6.png)



再点击左上角该按钮（Assemble the current file and clear breakpoints）

![Snipaste_2024-05-31_13-47-49.png](https://s2.loli.net/2024/05/31/grohIO94wVXd7Hi.png)



会显示如下界面

![Snipaste_2024-05-31_13-49-33.png](https://s2.loli.net/2024/05/31/QFTZz9o7vNk3JMe.png)



再点击左上角该按钮（Run the current program）

![Snipaste_2024-05-31_13-50-09.png](https://s2.loli.net/2024/05/31/s8Lt7OZuQ642TBh.png)



控制台成功输出，软件开始运行。

![Snipaste_2024-05-31_13-51-06.png](https://s2.loli.net/2024/05/31/buVX4UYazGZ6r1p.png)



再根据设计文档中的测试部分进行测试即可。
